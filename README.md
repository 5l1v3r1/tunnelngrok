# Como usar | How to use
Como instalar:
wget https://github.com/fellipmelo/tunnelngrok.git
cd tunnelngrok 
chmod 770 *
./tunnelngrok.py

para executar novamente basta digitar no terminal 'tn.py'

# Desinstalar

cd /etc/tunnelngrok/ <br/>
./uninstall.sh
